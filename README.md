# ⚡ PROMPTly

PROMPTly is an AI-powered prompt engineering assistant that transforms vague prompts into structured, high-quality prompts.

## Features

- Chat-style UI
- Adaptive follow-up questions
- Prompt templates
- Human-like output mode
- Cheatcode buttons
- Prompt scoring
- Prompt history
- Download & share prompts

## Tech Stack

- Streamlit
- Gemini API

## Setup

1. Clone repo
2. Add API key in `.streamlit/secrets.toml`
